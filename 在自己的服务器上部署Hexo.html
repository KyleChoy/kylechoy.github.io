发表于 2022-03-06 分类于 网站

此前写了一篇 Windows 下使用 Github 搭建免费个人博客 的文章，但是在国内使用 Github Pages 的体验实在是一般，而 Gitee Pages 的自动化部署又需要付费，因此参考网上资料，通过在自己的服务器上搭建 Git 服务并在 Virtualmin 的环境下开启 Web 服务。
